# KaushalHub

## Overview

KaushalHub is a full-stack freelance marketplace platform that connects clients with freelancers and skilled professionals. Clients can create projects, receive proposals, and manage their projects, while freelancers can browse projects and submit proposals.

This project was built as a portfolio project to gain hands-on experience with full-stack web development using the MERN stack.

---

### Authentication & Authorization

* User Registration

* Role-based Access Control (Client / Freelancer)

* Create Projects
* View Own Projects
* Manage Project Proposals
